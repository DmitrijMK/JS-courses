## JS задачи
1. Реализовать функцию, которая суммирует аргументы.
```
function sum(x) {
  ...
}

sum(1)(2)(3) === 6
```

2. Реализовать функцию, которая суммирует аргументы, если аргумент не передан - вернуть сумму.
```
function sum(x) {
  ...
}

sum(1)(2)(3)...(N)() === сумме всех чисел до N
```
